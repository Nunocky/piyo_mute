# ぴよ日記ミュート

https://piyo.hatelabo.jp/ で特定のタグの投稿を非表示にするための Google Chrome 拡張です。

## 仕様

- ツールバーの横にアイコンが表示される。アイコンをクリックすると、設定画面が表示される
  - 設定画面からミュートしたいタグを入力することができる（1行に1タグ、# は不要）
  - 設定を保存すると、ページをリロードせずに即座にフィルターが適用される

## Chrome Web Store ストアリスティング情報

### 拡張機能名

ぴよ日記ミュート

### カテゴリ

ソーシャルネットワーク

### 概要（132文字以内）

piyo.hatelabo.jp で見たくないタグの投稿を非表示にします。ミュートしたいタグを登録するだけで、ページをリロードせず即座にフィルターが適用されます。

### 詳細な説明

piyo.hatelabo.jp（ぴよ日記）で、特定のタグが付いた投稿を非表示にできる拡張機能です。

### 主な機能

- ミュートしたいタグをテキストで登録（1行に1タグ、# 不要）
- 設定を保存した瞬間にフィルターが適用、ページのリロード不要
- 設定は Chrome の同期ストレージに保存されるため、複数デバイス間で共有可能

### 使い方

1. ツールバーのアイコンをクリックして設定画面を開く
2. 非表示にしたいタグを1行ずつ入力する
3. 「保存」ボタンをクリックするだけで完了

## Chrome Web Store 提出用 ZIP の作成

```bash
npm run build
cd dist && zip -r ../piyo_mute.zip . && cd ..
```

`piyo_mute.zip` がプロジェクトルートに生成されます。`manifest.json` が ZIP のルートに含まれる形になります。

## インストール手順(開発者向け)

- `npm run build` を実行して、dist/ フォルダを生成する
- Chrome で chrome://extensions を開く
- 右上の 「デベロッパーモード」をオン
- 「パッケージ化されていない拡張機能を読み込む」 をクリック
- dist/ フォルダを選択
