# ぴよ日記フィルタ

https://piyo.hatelabo.jp/ で特定のユーザー、または特定のタグの発言を非表示にするための google chrome拡張です。

## 仕様

- ツールバーの横にアイコンが表示される。アイコンをクリックすると、フィルタの設定画面が表示される
  - 設定画面から「ユーザー」「タグ」をタブとして用意する。それぞれのタブ内でミュートしたいユーザー名、またはタグを入力することができる。

## インストール手順(開発者向け)

- `npm run build` を実行して、dist/ フォルダを生成する
- Chrome で chrome://extensions を開く
- 右上の 「デベロッパーモード」をオン
- 「パッケージ化されていない拡張機能を読み込む」 をクリック
- dist/ フォルダを選択
